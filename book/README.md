# Multi-Signature Wallet (Stacks / Clarity)

## Overview
This repository contains a Clarity smart contract for a multi-signature wallet on the Stacks blockchain.

## Prerequisites
- [Clarinet CLI](https://github.com/hirosystems/clarinet)
- [Rust and Cargo](https://www.rust-lang.org/tools/install)
- Node.js (for optional frontend)
